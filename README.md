# TalentBridge  - Backend (PHP + MySQL)

## Setup
1. Create a database:
   CREATE DATABASE football_agent_db;

2. Import tables and sample data:
   - Run create_tables.php
   - Run insert_sample_data.php

3. Test endpoints:
   - get_users.php → view all users
   - add_user.php → create new user
   - update_user.php?id=1 → update user
   - delete_user.php?id=1 → delete user
